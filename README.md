# AI-Based-Student-Performance-Analytics-System

## Overview
The AI-Based Student Performance Analytics System is a Machine Learning-based application designed to analyze student academic records and predict future performance.

## Features
- Student Management
- Attendance Tracking
- Marks Analysis
- Performance Prediction
- Analytics Dashboard
- Report Generation

## Technologies Used
- Java
- Spring Boot
- MySQL
- Python
- Scikit-Learn
- React (Optional)

## Objectives
- Analyze student performance
- Predict future academic outcomes
- Identify at-risk students
- Generate reports
